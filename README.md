# 上海海事大学 统一认证平台 验证码识别模型训练(PyTorch)

## 技术栈

- Python
- PyTorch
- OpenCV
- ResNet

## 操作步骤

- 图像转换为灰度图
- 图像二值化
- 对图像7，3分最后一部分作为“等于”与“=”的区分标准
- 使用ResNet-18对最后的符号进行区分，分为3部分(数字，运算符，数字)
- 对1,3部分使用ResNet-18进行**数字**识别
- 对2部分使用ResNet-18进行**运算符**识别
